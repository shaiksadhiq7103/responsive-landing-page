# Responsive Landing Page with Interactive Navigation Menu

A modern and responsive landing page featuring a fixed navigation bar that dynamically changes its appearance when users scroll through the page and interact with menu items.

## 🚀 Features

- Fixed navigation menu visible across all sections
- Navbar style changes on page scroll
- Interactive hover effects on menu items
- Fully responsive design for desktop, tablet, and mobile devices
- Smooth transitions and animations
- Clean and modern UI layout
- Optimized HTML, CSS, and JavaScript structure

## 📸 Preview

The landing page includes:

- Responsive hero section
- Interactive navigation bar
- Scroll-based navbar effects
- Hover animations on navigation links
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
